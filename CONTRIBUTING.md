# Contributing to OmniToken

We welcome contributions to OmniToken!

## Getting Started

- Fork the repository.
- Clone your fork: `git clone https://github.com/YOUR_USERNAME/omnitoken.git`
- Create a new branch: `git checkout -b my-feature-branch`
- Make your changes.
- Run the tests: `cargo test` and `pytest`
- Push to your fork: `git push origin my-feature-branch`
- Open a pull request.

## Code Style

- Please follow the existing code style.
- We use `clippy` for Rust and `black` for Python.
- Run `cargo clippy` and `black .` before committing.

## Pull Requests

- Please include a clear description of your changes.
- Make sure all tests pass.
- Link to any relevant issues.
